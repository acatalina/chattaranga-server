Welcome to Chattaranga API

Find below instructions to interact with our API.

  • GET /translate?text=(text)&sourceLangauge=(en)&targetLanguage=(sp)

    Returns the translated text. The translation is done from sourceLangauge to targetLanguage

  • GET /badges

    gets all available badges

  • GET /:languageId/:levelId/prompts

    gets all available prompts

  • GET /users/:username

    gets a particular user's profile

  • GET /languages

    gets all available languages

  • GET /levels

    gets all available levels

  • PUT /users/:username/smileys

    updates a particular user's smileys

  • PUT /users/:username/:language

    send a valid query (teacherpoints, numofchats or talktime) and value to update a particular language profile of a username.

  • POST /users

    creates a new user: Please send a raw JSON object with key-value pairs of username in lowercase, name, email and userlanguages (array with objects where every object has a key of language ID and value of level ID.
